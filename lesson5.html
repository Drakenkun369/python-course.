Lesson 5: Loops (for, while)

ðŸ“˜ Explanation

Loops let you execute a block of code multiple times. Python supports two types of loops:

  • for loop - used to iterate over a sequence (like a list or range).
  • while loop - runs as long as a condition is true.

Use break to exit a loop early and continue to skip the rest of the current loop iteration.
