Control flow allows you to make decisions in your code using conditions. The main statements are:

  • if - Checks a condition
  • elif - Checks another condition if the first is False
  • else - Runs if none of the conditions are True

Indentation is important in Python; all blocks of code under if/elif/else must be indented.
